Default HLWM Keybindings:

Note: hlwm Mod is defined as mod4; mod4 key = Windows or Super key.

Super+Shift+q

Super+Shift+r
Reload

Super+Shift+c
Close

Super+Return
Terminal

Super+d
Menu bar

Super+o
splits the screen horizontally
